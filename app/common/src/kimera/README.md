# Kimera Library

Kimera is a Python library that provides shared functionality and utilities for the Kimera project. Kimera is a platform that leverages various technologies such as OpenAI API, Kafka, Celery, MySQL Connector, PyMongo, and MongoEngine to uphold Kimera based Microservices.

## Features

- **OpenAI Integration**: Seamless integration with the OpenAI API for natural language processing tasks.
- **Kafka Support**: Simplifies interaction with Kafka for message streaming and processing.
- **Celery Integration**: Facilitates task scheduling and distributed task execution using Celery.
- **Database Connectivity**: Provides wrappers and utilities for interacting with MySQL using MySQL Connector, and MongoDB using PyMongo and MongoEngine.

## Scope

mandatory submodule for any Kimera Microservice
